## ğŸš€ Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager
- Windows, macOS, or Linux operating system

### Installation

1. Clone the repository:
```bash
git clone https://github.com/AnukarOP/Water-Reminder-desktop-app.git
cd Water-Reminder-desktop-app
```

2. Install dependencies:
```bash
npm install
```

3. Start the application:
```bash
npm start
```

### Building from Source

To create a distributable package for your platform:

```bash
npm run build
```

The compiled application will be available in the `dist` folder.
